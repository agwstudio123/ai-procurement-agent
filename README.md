# 🏗️ BuildProcure AI

**AI-Powered Construction Procurement Platform built on Arc**

BuildProcure AI is an intelligent procurement platform that simplifies construction material sourcing, supplier management, and USDC payments.

Instead of manually searching for suppliers, comparing prices, and managing procurement through phone calls and spreadsheets, contractors can submit their procurement requirements and let AI recommend the best suppliers.

The platform combines **Artificial Intelligence**, **Arc blockchain**, **Circle USDC**, and **smart contracts** to create a transparent, efficient, and secure procurement workflow.

---

##  Problem

Construction procurement is often:

- Time-consuming
- Manual
- Fragmented
- Difficult to track
- Prone to payment delays

Contractors spend significant time:

- Searching for suppliers
- Comparing material prices
- Managing procurement requests
- Tracking orders
- Coordinating supplier payments

---

# 💡 Solution

BuildProcure AI streamlines procurement by providing:

- 🤖 AI-powered supplier recommendations
- 📋 Bill of Quantities (BOQ) submission
- 💰 Material price comparison
- 📦 Procurement management
- 🏪 Supplier registration & dashboards
- 📦 Order management & tracking
- 💵 USDC payment workflow
- 🔗 Arc smart contract integration

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/agwstudio123/ai-procurement-agent.git
```

Move into the project

```bash
cd ai-procurement-agent
```

Install dependencies

```bash
npm install
```

Run backend

```bash
npm run dev
```

Run frontend

```bash
cd client
npm install
npm run dev
```

---

# 🔐 Environment Variables

Create a `.env` file.

Example:

```env
MONGODB_URI=<your_mongodb_uri>
PRIVATE_KEY=<your_private_key>
ARC_RPC_URL=<your_arc_rpc_url>
USDC_CONTRACT=<usdc_contract_address>
CONTRACT_ADDRESS=<your_contract_address>
```
